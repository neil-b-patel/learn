Buckle up. It's time to publish a website!
While this won't look the prettiest, it's still impressive! Also, you'll be prepared to make more websites (or make this one better).
1. Install Git on your Shell. (Mac: just type "git", Windows: GitBash come with it)
2. Make a new directory and a file called "index.html" in it.
3. Make a new GitHub repo, and push the repo to Github.
4. Go the settings of your repo, and deploy your Github Page.
