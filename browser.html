Let's begin with that crusty browser of yours.
  1. Choose a modern browser.
    • Google Chrome
    • Mozilla Firefox
    • Opera
    • Brave
  2. Install useful extensions.
    • Ad Blocker: AdBlock
    • Security: HTTPS Everywhere
    • Password Manager: LastPass
    • RAM Saver: The Great Suspender
    • Tab Saver: OneTab
  3. Learn keyboard shortcuts. You'll be suprised at how much this increases your productivity. I've listed my favs below. No need to learn them all at once. Start with things you do often like opening new tabs, closing tabs, searching, go back and forward, and switching tabs. Not sure what you should learn? Try the shortcuts with * in front of them!
    • *Ctrl + C : Copy selected text
    • *Ctrl + X : Cut selected text
    • *Ctrl + V : Paste copied/cut text
    • *Ctrl + T : Open a new tab
    • Ctrl + N : Open a new window
    • Ctrl + Shift + N : Open an incognito window
    • *Ctrl + L : Search in address bar
    • *Ctrl + Shift + T : Restore a closed tab
    • *Ctrl + W : Close a tab
    • Alt + F4 : Close a window
    • *Alt + Left Arrow : Go back
    • *Alt + Right Arrow : Go forward
    • Ctrl + R : Refresh
    • Ctrl + S : Save as
    • Ctrl + P : Print
    • *Ctrl + F : Find
    • Ctrl + O Open
    • F12 : Open developer tools
    • *Ctrl + Shift + J : Open console
    • Ctrl + H : Open history
    • Ctrl + J : Open downloads
    • Ctrl + U : Open source code
    • Ctrl + D : Bookmark
    • Ctrl + # (1 to 8) : Switch to chosen tab
    • Ctrl + 9 : Switch to last tab
    • *Ctrl + Tab : Switch to next tab
    • *Ctrl + Shift + Tab : Switch to previous tab
    • Ctrl (+) : Zoom in
    • Ctrl (-) : Zoom out
    • F11 : Fullscreen
